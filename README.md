# Study Streak

Study Streak is a web application that helps students build consistent study habits by tracking daily study sessions, maintaining streaks, and visualizing learning progress.

## Problem

Many students struggle to maintain a consistent study routine. Study Streak encourages regular learning through streak tracking and progress monitoring.

## Features

- User registration and login
- Daily study check-in
- Current study streak tracking
- Longest streak record
- Progress dashboard
- Study history tracking

## Core Screens

### Home Page
- Project introduction
- Login/Register buttons

### Login Page
- User authentication

### Register Page
- New account creation

### Dashboard
- Current streak
- Longest streak
- Progress summary

### Streak Page
- Daily study check-in
- Streak statistics
